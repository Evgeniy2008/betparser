# Betparser (PHP-only)

Проект полностью переведён на **PHP** и использует **The Odds API v4**.

## Что делает

- Берёт коэффициенты из `https://api.the-odds-api.com/v4/sports/upcoming/odds/`.
- Формирует пары:
  - 1-й букмекер всегда **Pinnacle**
  - 2-й букмекер — каждый доступный букмекер из ответа API
- Считает те же формулы, что были раньше:
  - футбол: 6 формул (3-way)
  - баскетбол/теннис: 2 формулы (2-way)
- Сохраняет результат в `data/merged_matches.json`.

## Файлы

- `index.php` — интерфейс и просмотр матчей
- `fetch.php` — синхронизация данных с The Odds API
- `data/merged_matches.json` — кэш матчей и коэффициентов

## Запуск

Открыть в браузере:

- `http://betparser/index.php`

Ручное обновление:

- Кнопка «Обновить данные» на странице
- или вызов: `http://betparser/fetch.php?force=1`

## API-ключ

Ключ The Odds API зашит в `fetch.php` (константа `ODDS_API_KEY`).

## Важно

- Node.js и логика Parimatch полностью удалены из проекта.
- В проекте оставлены только файлы, реально используемые текущей PHP-версией.
